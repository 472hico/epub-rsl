# Contributing

Thank you for helping improve `epub-rsl`.

## Development

Requirements:

- Node.js 20 or newer
- npm 10 or newer

```bash
git clone https://github.com/472hico/epub-rsl.git
cd epub-rsl
npm install
npm run check
```

Please open an issue before starting a large change. Pull requests should include tests for changed
behavior and should keep source EPUB files unmodified by default.

## Reporting security issues

Do not open a public issue for a vulnerability that could expose protected publications. Use
GitHub's private vulnerability reporting feature instead.
